Reações Químicas Reversíveis

As reações químicas reversíveis são aquelas que podem ocorrer tanto no sentido direto como no sentido inverso, ou seja, pode ser revertidas.

Por exemplo,

2A+B↔C2A + B \leftrightarrow C

pode ser caracterizada pela seguinte relação de equilíbrio:

K=ccca2cbK = \frac{c_c}{c_a^2 c_b}

cic_i representa a concentração do elemento químico ii.

Vamos definir uma variável xx representando o número de mols de CC produzidos. A conservação de massa pode ser usada para reformular a relação de equilíbrio como

K=cc,0+x(ca,0−2x)2(cb,0−x)K = \frac{c_{c,0} + x}{(c_{a,0} - 2x)^2(c_{b,0} - x)}

onde o subscrito 0 representa a concetração inicial de cada elemento.

Dado o valor de KK, ca,0c_{a,0},cb,0c_{b,0}, cc,0c_{c,0} determine o valor de xx considerando o chute inicial de xlx_l e xux_u e com nn algarismos significativos, ou seja, um intervalo (a,b) possui uma solução aproximada se |b−a||b-a| é menor que 0.5×10−n0.5 \times 10^{-n}

Entrada

A entrada é composta por um linha com 7 valores KK, ca,0c_{a,0},cb,0c_{b,0}, cc,0c_{c,0},xlx_l, xux_u e ε\varepsilon representando constante de equílibrio, concetração inicial dos elementos A, B, C, intervalo inicial e final para o número de mols produzidos de CC e o limite do erro aceito.

Saída

A saída é composta de uma única linha contendo o número de mols produzidos de C com n casas decimais. A solução deve ser truncada para n casas decimais.

Utilize a seguinte função:

def truncate(f, n):
    '''Truncates/pads a float f to n decimal places without rounding'''
    s = '%.12f' % f
    i, p, d = s.partition('.')
    return '.'.join([i, (d+'0'*n)[:n]])

15.92
